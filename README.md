# Starsailor

Starsailor is a [Saga](https://github.com/justintadlock/saga) child theme built for [Starsailor.it](http://starsailor.it) website.

## Requirements

* PHP >= 5.5
* Wordpress >= 4.3.0
* Saga >= 1.0.2
* Solero >= 1.1.0
